Help+Manual 7 Pro — это самый популярный программный комплекс для создания справки, пользовательской и различной технической документации. Он предназначен для разработки и локализации профессиональных справочных систем, руководств пользователя, баз знаний, электронных книг и другой документации в форматах CHM, WebHelp, PDF, DOCX, EPUB, MOBI и др. для настольных и мобильных приложений, iOS и Android, Windows и Linux. Что такое Help&Manual    

 Help&Manual  — это программный комплекс для разработки справки и

     другой технической документации в операционных системах Windows. В его состав входят:

- визуальный XML-редактор, работать с которым не сложнее, чем с MS Word;

- средство создания скриншотов,

- Screen Capture Tool;

- редактор рисунков, Impict Image Editor;

- редактор шаблонов для pdf-документов,

- Print Manual Designer;

- локальный сервер HM2GO для тестирования WebHelp;

- средства проверки правописания, создания контекстной справки и др.

Как с этим работать

Работа с Help+Manual не требует каких-либо специальных знаний, в том числе в области программирования. Для создания контента и шаблонов PDF-документов используются визуальные редакторы. Стандартный внешний вид выходных документов можно задать при помощи хорошо продуманного пользовательского интерфейса. Русифицировать и адаптировать готовые шаблоны под пользователя / заказчика / фирменный стиль можно, обладая лишь базовыми знаниями HTML и CSS. И только для того, чтобы разработать свои шаблоны, необходимо уверенно владеть HTML, CSS и JavaScript.

Освоить Help+Manual может абсолютно любой человек, желающий научиться быстро и эффективно создавать:

справку, встроенную в приложения ОС Windows, в формате CHM;

•справочные системы в формате WebHelp с адаптивным дизайном, работающие на устройствах под управлением Windows, Linux, iOS и Android, в т.ч. не требующие веб-сервера;

•техническую и пользовательскую документацию в формате PDF, которую можно использовать как в печатном, так и в электронном виде;

•техническую и пользовательскую документацию в формате MS Word;

•профессиональные электронные книги в формате EPUB для iPad, iPhone и других мобильных устройств;

•профессиональные электронные книги в формате MOBI для Amazon.
